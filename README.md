# Sweet Shop App 🍪

A modern, high-performance Flutter application for a Sweet/Cookie shop, designed with a focus on **Clean Code**, **Modular Architecture**, and a premium **User Experience**.

This project represents the **first milestone** in my professional Flutter learning journey. It demonstrates my ability to build scalable mobile applications and my commitment to following industry best practices from day one.

---

## 🚀 Why this project?
As an aspiring Flutter Developer, I built this app to bridge the gap between learning and professional implementation. My goal was to create a codebase that is:
- **Readable:** Following the "Clean Code" principles.
- **Maintainable:** Using a feature-based folder structure.
- **Scalable:** Leveraging robust State Management (BLoC).

---

## ✨ Key Features
- **Modern UI/UX:** A clean, appetizing design using `Google Fonts` and `Animate Do` for smooth transitions.
- **State Management:** Fully powered by `Flutter BLoC` (Cubit) for efficient UI updates and logic separation.
- **Map Integration:** Integrated `Flutter Map` with custom markers (configured for Heliopolis/Nasr City, Cairo).
- **Dynamic Home Screen:** Includes a `Carousel Slider` for featured products and `Smooth Page Indicators`.
- **Theme Support:** Managed theme states via BLoC.
- **Clean Assets Management:** Using `Flutter Gen` for type-safe asset access.

---

## 🏗️ Architecture & Folder Structure
The project follows a **Feature-based Clean Architecture** approach to ensure separation of concerns:

```text
lib/
├── core/            # Global widgets, themes, and utilities
│   ├── theme/       # App styling and colors
│   └── widgets/     # Reusable UI components (like FlutterMapWidget)
├── features/        # Independent modules
│   └── home_feature/
│       ├── data/          # Sample data and data sources
│       └── presentation/  # UI (Screens/Widgets) & BLoC (Logic)
└── main.dart        # Entry point
```

---

## 🛠️ Tech Stack & Tools
- **Framework:** [Flutter](https://flutter.dev)
- **State Management:** `flutter_bloc`
- **Navigation & UI:** `carousel_slider`, `smooth_page_indicator`, `readmore`, `dotted_border`.
- **Maps:** `flutter_map` & `latlong2`.
- **Animations:** `animate_do`.
- **Code Generation:** `flutter_gen` & `build_runner`.
